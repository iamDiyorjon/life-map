# Hayot xaritasi — Life Map

Interactive bilingual (UZ/EN) life-roadmap self-assessment.

## Run locally
```bash
npm install
npm run dev
```

## Deploy on Vercel
1. Push this folder to a GitHub repo.
2. vercel.com → Add New → Project → Import the repo.
3. Vercel auto-detects Vite (Build: `npm run build`, Output: `dist`). Click Deploy.

Built with React, Vite, Recharts, lucide-react.
